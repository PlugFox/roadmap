# Flutter Developer Roadmap

## How to start

Install dart

```bash
brew tap dart-lang/dart
brew install dart
```

Download dependencies and build the project

```bash
dart pub get
dart pub run build_runner build
dart pub global activate webdev
```

Serve the project

```bash
webdev serve
```
